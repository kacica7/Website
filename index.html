Hangman

This hangman game was a fun little project to work on in order to renew my C knowledge and take a break from uni problems. Check out hangman.c in my Finished-Projects repo on GitHub to check out the code and play the game! The gameplay is fairly simple, since I mainly focused on the loop, but I wanted to code it so that the the word that needs to be guessed is taken from a text file.
